# AWS DCO GenAI Portfolio

## 1. 과정명
AWS DCO 인턴십 직무 이해를 위한 생성형 AI 활용 및 포트폴리오 제작 과정

## 2. 포트폴리오 목적
이 저장소는 AWS DCO(Data Center Operations) 직무를 이해하기 위해, 생성형 AI를 학습 도구로 활용하며 진행한 수업 산출물을 정리한 **학습형 포트폴리오**입니다. 실제 AWS 인턴십 업무를 수행한 결과물이 아니라, 공개된 직무 맥락과 교육용 예시를 바탕으로 직무 용어·SOP·업무 흐름을 이해하는 연습 과정을 담고 있습니다.

## 3. 교육용 샘플 데이터 사용 안내
- 이 포트폴리오에 포함된 모든 로그, SOP, 장비명, 시나리오는 **교육용으로 만들어진 샘플 데이터**이며, 실제 AWS 내부 자료가 아닙니다.
- 장비명이나 파일명에는 `SAMPLE`, `DEMO`, `EDU` 접두어를 사용하여 교육용 데이터임을 표시합니다.
- IP 주소가 필요한 경우 문서화 예시용 주소(`192.0.2.x`, `198.51.100.x`, `203.0.113.x`)만 사용합니다.
- 모든 분석 결과는 "확인이 필요한 관찰 내용" 수준으로 작성되며, 실제 장애 원인을 단정하지 않습니다.

## 4. 각 폴더의 역할

| 폴더 | 역할 |
|---|---|
| `01_dco_glossary/` | DCO 직무 이해에 필요한 인프라·업무 용어(Rack, NIC, SOP, Ticket, Escalation 등)를 정리한 용어 사전 |
| `02_dco_profile/` | DCO 직무의 정의, 주요 업무, 협업 유형 등 직무 이해와 관련된 자기 분석 자료 |
| `03_sop_analysis/` | 교육용 샘플 SOP를 읽고 목적·절차·주의사항으로 분석하거나 체크리스트로 변환한 자료 |
| `04_environment_setup/` | 실습에 필요한 개발/학습 환경 설정 과정을 정리한 자료 |
| `05_antigravity_cli/` | CLI 도구 학습 및 실습 관련 자료 (교육용 실습 목적) |
| `06_cli_file_automation/` | CLI를 활용한 파일 처리 자동화 실습 자료 |
| `07_log_analysis_script/` | 교육용 샘플 로그를 분석하는 스크립트 및 실습 결과 |
| `08_incident_analysis/` | 교육용 샘플 장애 시나리오를 분석한 연습 자료 |
| `09_incident_report/` | 교육용 샘플 시나리오 기반 Incident Report 작성 연습 자료 |
| `10_stitch_dco_dashboard/` | DCO 관련 데이터를 시각화한 학습용 대시보드 자료 |
| `11_exam_prep/` | 지필시험 대비를 위한 용어 요약 및 정리 자료 |

*(각 폴더의 세부 내용은 폴더 안의 README나 문서를 통해 추가로 확인할 수 있습니다.)*

## 5. 보안상 포함하지 않는 정보
아래 정보는 이 포트폴리오 어디에도 포함되지 않습니다.

- 실제 AWS 내부 절차, 실제 사내 문서
- 실제 장비명, 실제 장비 시리얼 번호
- 실제 티켓 번호, 실제 고객 정보
- 실제 IP 주소, 실제 계정 정보(로그인 정보, 자격 증명 등)
- 실제 장애 사건에 대한 원인 단정 서술
